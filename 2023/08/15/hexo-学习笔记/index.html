hexo 学习笔记

hexo命令

  • hexo new “标题” #创建新的文章
  • hexo clean #清理
  • hexo g #生成静态页面
  • hexo s #启动本地博客服务
  • http://localhost:4000/ #本地访问地址
  • ctrl+c #结束本地博客服务
  • hexo d #部署到远程Github仓库

python实现自动推送到Git

其实就是把几个命令行一起写在python中

import os
# 切换到博客目录（根据实际路径进行修改）
blog_path = "实际路径"
os.chdir(blog_path)

# 清理
print("--------------------正在清理中--------------------")
os.system("hexo clean")

# 生成静态文件
print("--------------------生成静态文件中--------------------")
os.system("hexo generate")

# 部署到线上
print("--------------------正在部署到Git中--------------------")
os.system("hexo deploy")

# 等待用户按下回车键
input("按回车键退出...")
